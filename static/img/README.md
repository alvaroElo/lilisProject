# Imágenes Pendientes

Coloca las siguientes imágenes en este directorio:

1. **Logo-lilis-header.png**
   - Logo para el header (fondo rojo #D20A11)
   - Tamaño recomendado: 200px altura
   - Formato: PNG con fondo transparente

2. **Logo-lilis-footer.png**
   - Logo para el footer (fondo oscuro #230E00)
   - Tamaño recomendado: 150px altura
   - Formato: PNG con fondo transparente

3. **Logo-lilis.png**
   - Logo principal para login y documentos
   - Tamaño recomendado: 300px altura
   - Formato: PNG con fondo transparente

## Nota Temporal

Mientras no tengas los logos, puedes usar placeholders o texto.
El diseño está preparado para que al agregar las imágenes,
se vean correctamente sin necesidad de modificar el código.
